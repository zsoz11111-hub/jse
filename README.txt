우리 둘 V5 적용 순서

1. setup_v5.sql을 Supabase SQL Editor에서 실행합니다.
2. config.js의 Publishable key가 기존처럼 들어 있는지 확인합니다.
3. GitHub 저장소 파일을 V5 파일로 교체합니다.
4. 기존 홈 화면 아이콘은 삭제한 뒤 다시 추가하는 것을 권장합니다.

가계부 바로가기
- 일반 아이콘을 누르면 홈 화면으로 들어갑니다.
- Android/갤럭시에서는 앱 아이콘을 길게 눌러 '빠른 지출 입력'을 선택할 수 있습니다.
- 별도의 지출 입력 전용 아이콘이 필요하면 아래 주소를 Safari/Chrome에서 열고 홈 화면에 추가하세요.
  https://본인주소.github.io/jse/?quick=expense

아이폰
- Safari에서 위 빠른 입력 주소 열기
- 공유 > 홈 화면에 추가
- 이름을 '지출 입력'으로 지정

갤럭시
- Chrome에서 위 빠른 입력 주소 열기
- 메뉴 > 홈 화면에 추가 또는 앱 설치
